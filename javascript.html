JavaScript page

This covers all the hard but important stuff you should know when learning JavaScript.

  • Audience: Intermediate JS developers.
  • Goal: Understand how JavaScript works, what's going on "under the hood".

Pre-requisites:

  1. Install NodeJS.
  2. Install http-server by running the following in your terminal

    npm install http-server -g

Before starting:

  1. Create a training folder.
  2. Download and unzip the js-exercises folder into your training folder.
