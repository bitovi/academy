Jira Account Setup page

Learn how to set up a new Jira Account and create some initial projects.

Overview

This training requires a Jira Premium Plan and several company-managed Scrum projects. Fortunately, Jira provides free accounts for 30 days. We recommend doing this training in a free training instance.

This page walks through setting up the correct Premium Plan and creating company-managed Scrum Projects.
